Using Spring Beans

On NixMash: Using Spring Beans in Wicket

Returning text from a Spring Service using the @SpringBean annotation, Wicket's equivalent of @AutoWired.

[serviceText]
>// TextService Spring Service Class

@Service
public class TextService {
    public String getText() {
        return "delivered by spring service";
    }
}

// DemoPage Class

@SpringBean
private TextService textService;

 /* spring integration in wicket component */
add(new Label("textService", String.format("From Spring Example Service: %s",
        textService.getText())));
